# Dataset Card — Air Quality Dataset

## 1. Источник данных
Данные собраны из открытых наблюдений за качеством воздуха.  
Набор содержит ежедневные измерения различных загрязнителей в городской среде.

Формат данных адаптирован для проекта по предсказанию уровня загрязнения воздуха.

---

## 2. Структура набора данных

После предобработки датасет содержит **13 признаков**, включая:

| Признак      | Описание |
|--------------|----------|
| `PM2.5`      | Мелкодисперсная пыль (µg/m³) |
| `PM10`       | Крупнодисперсная пыль (µg/m³) |
| `NO`         | Оксид азота |
| `NO2`        | Диоксид азота |
| `NOx`        | Оксиды азота суммарно |
| `NH3`        | Аммиак |
| `CO`         | Угарный газ |
| `SO2`        | Диоксид серы |
| `O3`         | Озон |
| `Benzene`    | Бензол |
| `Toluene`    | Толуол |
| `Xylene`     | Ксилол |
| `target`     | Итоговый показатель качества воздуха (AQI), который мы предсказываем |

Размер датасета: **6236 строк × 13 признаков**.

---

## 3. Предобработка

В `data/raw/prepare.py` выполняются:

- удаление пропусков;
- переименование столбцов в корректный формат;
- фильтрация некорректных значений;
- сохранение в `data/processed/air_quality.csv`.

Целевой признак — `target`.

---

## 4. Использование

Датасет применяется для:

- обучения модели регрессии,
- оценки метрил (MSE),
- инференса в FastAPI сервисе (`/predict` endpoint).

---

## 5. Ограничения и замечания

- Данные могут содержать выбросы из-за особенностей замеров датчиков.
- Датасет не подходит для задач классификации.
- Значения измерены в разные дни и могут иметь сезонность, не учтённую в текущей версии модели.
